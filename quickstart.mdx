## Get Started in Minutes

This quickstart guide will get you up and running with the Miniswimmer swimming academy platform in just a few steps.

<Note>
  For a more detailed installation with all configuration options, see our [complete installation guide](/setup/installation).
</Note>

## Prerequisites

Before you begin, make sure you have:

- Node.js 14+ installed
- PostgreSQL database
- Git

## Quick Setup

<Steps>
  <Step title="Clone the Repository">
    Get the source code:
    
    ```bash
    git clone https://github.com/mortegac/miniswimmer-cl-public-site.git
    cd miniswimmer-cl-public-site
    ```
  </Step>

  <Step title="Install Dependencies">
    Install all required packages:
    
    <CodeGroup>
    ```bash npm
    npm install --legacy-peer-deps
    ```
    
    ```bash yarn
    yarn install
    ```
    </CodeGroup>
    
    <Warning>
      Use `--legacy-peer-deps` flag with npm due to React 19 compatibility requirements.
    </Warning>
  </Step>

  <Step title="Set Up Environment Variables">
    Create a `.env` file in the root directory:
    
    ```bash .env
    # Database
    DATABASE_URL="postgresql://user:password@localhost:5432/miniswimmer"
    SHADOW_DATABASE_URL="postgresql://user:password@localhost:5432/miniswimmer_shadow"
    
    # Prismic CMS
    NEXT_PUBLIC_PRISMIC_ENVIRONMENT="miniswimmerchile"
    PRISMIC_ACCESS_TOKEN="your-prismic-token"
    
    # EmailJS
    NEXT_PUBLIC_EMAILJS_SERVICE_ID="your-service-id"
    NEXT_PUBLIC_EMAILJS_PUBLIC_KEY="your-public-key"
    
    # Optional: Analytics
    NEXT_PUBLIC_GOOGLE_TAG_MANAGER_ID="your-gtm-id"
    NEXT_PUBLIC_GOOGLE_ANALYTICS_ID="your-ga-id"
    ```
    
    <Tip>
      See the [environment variables guide](/setup/environment-variables) for details on obtaining these values.
    </Tip>
  </Step>

  <Step title="Initialize Database">
    Set up your PostgreSQL database with Prisma:
    
    ```bash
    npx prisma generate
    npx prisma db push
    ```
    
    This creates all necessary tables based on the Prisma schema.
  </Step>

  <Step title="Start Development Server">
    Launch the development server:
    
    ```bash
    npm run dev
    ```
    
    The application will be available at [http://localhost:3000](http://localhost:3000)
  </Step>

  <Step title="Verify Installation">
    Open your browser to [http://localhost:3000](http://localhost:3000) and you should see:
    
    - The Miniswimmer homepage with hero section
    - Language switcher (English/Spanish) in the header
    - Responsive navigation menu
    - Footer with social links
    
    <Check>
      Success! Your Miniswimmer platform is now running locally.
    </Check>
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="npm install fails with peer dependency errors">
    Use the `--legacy-peer-deps` flag with npm:
    ```bash
    npm install --legacy-peer-deps
    ```
    This is required due to React 19 compatibility with some packages.
  </Accordion>
  
  <Accordion title="Database connection fails">
    Verify your `DATABASE_URL` is correct and PostgreSQL is running:
    ```bash
    psql -U your_user -d miniswimmer
    ```
    Make sure the database exists before running migrations.
  </Accordion>
  
  <Accordion title="Port 3000 is already in use">
    Specify a different port:
    ```bash
    PORT=3001 npm run dev
    ```
  </Accordion>
  
  <Accordion title="Images from Prismic not loading">
    Check that your `NEXT_PUBLIC_PRISMIC_ENVIRONMENT` matches your Prismic repository name, and that the image domains are configured in `next.config.js`.
  </Accordion>
</AccordionGroup>
