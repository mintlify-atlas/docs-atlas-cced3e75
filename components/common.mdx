## Overview

Common components are reusable UI building blocks used across multiple pages in the Miniswimmer platform. They are designed to be flexible, internationalized, and maintainable.

## Hero Component

The Hero component (`src/components/Common/Hero/index.tsx`) displays the main banner section with title, subtitle, CTA button, and hero image.

### Features

- Large heading with highlighted text
- Optional subtitle and CTA button
- Hero image with responsive sizing
- Decorative wave SVG at bottom
- Primary background color
- Fully responsive layout

### Usage

```tsx
import Hero from '@/components/Common/Hero';

<Hero pageTraslation="homepage" />
```

### Props

<ParamField path="pageTraslation" type="string" required>
  Translation namespace for the page (e.g., "homepage", "about")
</ParamField>

### Translation Structure

The Hero component expects these translation keys:

```json
{
  "homepage": {
    "hero_section": {
      "title": "Welcome to <marker>Miniswimmer</marker>",
      "subtitle": "Swimming lessons for all ages",
      "cta": "Book a Class",
      "url": "/contact",
      "image": "/images/hero/header-2.png"
    }
  }
}
```

              )}
            </div>
          </div>
        ))}
      </div>
    </section>
  );

```tsx CountUp Animation
// src/components/Common/Counter/CountUp.tsx
import React, { useEffect, useState, useRef } from 'react';

interface CountUpProps {
  targetNumber: number;
}

const CountUp: React.FC<CountUpProps> = ({ targetNumber }) => {
  const [count, setCount] = useState(0);
  const countRef = useRef<HTMLSpanElement>(null);

  const handleIntersection: IntersectionObserverCallback = entries => {
    const entry = entries[0];
    if (entry.isIntersecting) {
      startCount();
    }
  };

  const startCount = () => {
    let startTime: number | undefined;
    const duration = 1000;

    const animate = (time: number) => {
      if (!startTime) startTime = time;
      const progress = (time - startTime) / duration;
      const currentCount = Math.ceil(progress * targetNumber);

      setCount(currentCount > targetNumber ? targetNumber : currentCount);

      if (progress < 1 && count < targetNumber) {
        requestAnimationFrame(animate);
      }
    };

    requestAnimationFrame(animate);
  };

  useEffect(() => {
    const observer = new IntersectionObserver(handleIntersection);
    if (countRef.current) {
      observer.observe(countRef.current);
    }

    return () => {
      observer.disconnect();
    };
  }, []);

  return (
    <span className='counter mr-4 inline-block' ref={countRef}>
      {count}
    </span>
  );
};
```
</CodeGroup>

<Note>
  The CountUp component uses Intersection Observer API to trigger animation when scrolled into view.
</Note>

## FAQ Component

Accordion-style FAQ section (`src/components/Common/FAQ/index.tsx`).

### Features

- Collapsible Q&A items
- Smooth expand/collapse animation
- Active state tracking
- Dark mode support

### Usage

```tsx
import FAQ from '@/components/Common/FAQ';

<FAQ />
```

### Implementation

```tsx FAQ.tsx
'use client';

import SectionHeader from '@/components/Common/SectionHeader';
import { useTranslations } from 'next-intl';
import { useState } from 'react';

const FAQ = () => {
  const [activeFaq, setActiveFaq] = useState<number | null>(0);
  const t = useTranslations('homepage.faq_section');
  const faq: Record<string, string>[] = t.raw('items');

  const handleFaqToggle = (id: number) => {
    activeFaq === id ? setActiveFaq(null) : setActiveFaq(id);
  };

  return (
    <section className='overflow-hidden bg-gray-1 py-17.5 dark:bg-black lg:py-22.5 xl:py-27.5'>
      <SectionHeader title={t('title')} description={t('subtitle')} />

      <div className='mx-auto w-full max-w-[662px] px-4 sm:px-8 xl:px-0'>
        <div className='flex flex-col gap-4'>
          {faq?.map(({ question, answer }, i) => (
            <div key={i} className='rounded-lg bg-white shadow-1 dark:bg-gray-dark'>
              <button
                onClick={() => handleFaqToggle(i)}
                className='text-medium flex w-full items-center justify-between px-6 py-4 text-left text-lg text-dark dark:text-white md:text-xl'
              >
                {question}
                <span className={`duration-300 ${activeFaq === i ? 'rotate-180' : ''}`}>
                  {/* Chevron SVG */}
                </span>
              </button>
              <div
                className='grid'
                style={{
                  gridTemplateRows: `${activeFaq === i ? '1fr' : '0fr'}`,
                  transition: 'grid-template-rows 300ms',
                }}
              >
                <div className='overflow-hidden transition-all duration-300'>
                  <p className='border-t border-stroke px-6 py-7 dark:border-stroke-dark'>
                    {answer}
                  </p>
                </div>
              </div>
            </div>
          ))}
        </div>
      </div>
    </section>
  );
};
```

<Tip>
  The grid-template-rows transition creates smooth accordion animation without JavaScript height calculations.
</Tip>

## Best Practices

1. **Always pass pageTraslation prop** for i18n support
2. **Use SectionHeader** for consistent section styling
3. **Leverage `<marker>` tags** for highlighted text in translations
4. **Optimize images** with Next.js Image component
5. **Test responsive behavior** on mobile, tablet, and desktop
6. **Maintain consistent spacing** using Tailwind utility classes
7. **Support dark mode** with dark: prefix classes
