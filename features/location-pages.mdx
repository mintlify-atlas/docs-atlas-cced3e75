## Overview

Miniswimmer serves multiple geographic locations across Chile and the United States, with dedicated landing pages for each city and region. These location-specific pages are optimized for local SEO and provide customized content for different markets.

## Supported Locations

### Chile Locations

The platform supports multiple cities in Chile:

- **Peñalolén** (`/clases-natacion-penalolen`)
- **Viña del Mar** (`/clases-natacion-vina-del-mar`)
- **Maintencillo** (`/clases-natacion-maintencillo`)
- **La Reina**
- **Nuñoa**
- **Vitacura**
- **Lo Barnechea**
- **Rancagua**
- **Valparaíso**
- **Chillán**

### United States Locations

In the US, Miniswimmer operates in:

- **Miami-Dade County** (`/swimming-lessons-in-miami-dade`)
- **Broward County** (`/swimming-lessons-in-broward`)

## Creating a New Location Page

<Steps>
  <Step title="Create Page File">
    Add a new file in `src/app/(site)/`:
    
    ```bash
    mkdir src/app/(site)/clases-natacion-new-city
    touch src/app/(site)/clases-natacion-new-city/page.tsx
    ```
  </Step>

  <Step title="Add Metadata">
    Define SEO metadata, Open Graph tags, and schema markup.
  </Step>

  <Step title="Create Component">
    Build the location component in `src/components/`:
    
    ```bash
    mkdir src/components/LessonsInNewCity
    touch src/components/LessonsInNewCity/index.tsx
    ```
  </Step>

  <Step title="Add Translations">
    Add location-specific content to `dictionary/es.json` and `dictionary/en.json`.
  </Step>

  <Step title="Configure Routes">
    Ensure Next.js routing picks up the new page (automatic with file-based routing).
  </Step>

  <Step title="Add to Navigation">
    Update header navigation if needed in `src/components/Header/`.
  </Step>
</Steps>

## Best Practices

<Tip>
  **Localize everything**: Phone numbers, addresses, testimonials, and images should all be location-specific for maximum relevance.
</Tip>

- **Use local keywords**: Include city names, neighborhoods, and region-specific terms
- **Unique content per location**: Avoid duplicating content across pages
- **Local phone numbers**: Use region-appropriate WhatsApp numbers
- **Location-specific testimonials**: Show reviews from families in that area
- **Local landmarks**: Reference nearby areas, neighborhoods, or landmarks
- **Optimize for mobile**: Many parents search for swim lessons on mobile devices
- **Fast loading**: Use Prismic's image optimization for quick page loads
