## What is Miniswimmer?

Miniswimmer is a Next.js-based SaaS platform built specifically for swimming academies. It provides everything you need to run a modern swimming school:

- **Multi-location support** - Manage multiple swimming school locations with location-specific pages
- **Bilingual interface** - Full English and Spanish language support with next-intl
- **Student management** - Authentication and session management powered by Prisma
- **Class scheduling** - Built-in scheduling and rescheduling system for classes
- **Content management** - Seamless integration with Prismic CMS for easy content updates
- **Modern tech stack** - Built with Next.js 15, React 19, TypeScript, and Tailwind CSS

## Built With Modern Technologies

Miniswimmer leverages cutting-edge web technologies:

- **Next.js 15** - React framework with App Router for optimal performance
- **React 19** - Latest React features for modern UI development
- **TypeScript** - Type-safe development for reliability
- **Prisma** - Modern database ORM for PostgreSQL
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **next-intl** - Internationalization library for multi-language support
- **React Hook Form + Yup** - Powerful form handling with validation
- **EmailJS** - Email notifications for inquiries and bookings
- **Prismic CMS** - Headless CMS for content management

## Who is This For?

This platform is ideal for:

- **Swimming academies** operating multiple locations
- **Swim schools** needing bilingual support (English/Spanish)
- **Aquatic programs** requiring class scheduling and booking systems
- **Swimming instructors** wanting to showcase their teaching methodology
- **Educational swim programs** focused on water safety and child development
